## ğŸ§¾ Short Project Specification

**Project Title:** PopCorn

**Objective:**\
Build a responsive web application that displays a list of movies using a card-based UI while allowing users to search, filter, and view movie information in a clean and modern interface.

**Core Requirements:**

- Display movies dynamically using JavaScript
- Each movie card must show poster, title, release date, rating, genre, and short description
- Implement search by title and filter by genre
- Include a dark mode toggle
- Use a fixed navigation bar
- Ensure responsiveness with CSS media queries
- Deploy the project using GitHub Pages

**Target Users:**\
Beginner to intermediate users exploring frontend development concepts.
